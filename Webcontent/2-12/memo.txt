function안에서 return과 추가작업을 입력하면
return만 작업하고 추가 수행은 이뤄지지 않는다.
만약 return 앞에 기타작업이 있다면 이 작업은 수행된다
즉, return "까지만" 수행된다

return을 쓸 때 주의사항
한번 return을 하며 해당 함수는 끝난다.
그렇기 때문에 return 뒤에 작업을 한다면 출력되지 않는다.
(return 앞에 작업을 한다면 출력된다)

return은 말 그대로 다시 돌려준다는 의미로
해당 함수의 변수값을 지정해주면 해당 code를 안에서 실행해서 그 결과값을 해당 함수의 변수값을 선언한 변수한테 돌려주는 것
